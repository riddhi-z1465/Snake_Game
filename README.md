## 🎮 How to Play
- Use **Arrow Keys** to move the snake:
  - ⬆️ Up
  - ⬇️ Down
  - ⬅️ Left
  - ➡️ Right
- Eat the glowing squares to grow.
- Avoid colliding with yourself.
- The game wraps around the edges.
- Click the **Restart** button to start over.
